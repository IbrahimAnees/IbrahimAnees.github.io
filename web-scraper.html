Web Scraper

Website scraper which scrapes the CompareTV website to retrieve all Netflix movies/shows.

Tech stack: Java

Repo: https://github.com/ibrahimanees/website-scraper/

This scraper scrapes the following webpage: https://www.comparetv.com.au/streaming-search-library/?ctvcp=1770/

The scraper returns the total number of Netflix shows/movies in Australia, as well as the title and URL of each show.
