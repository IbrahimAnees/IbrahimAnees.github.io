Sync App

Syncs two given directories.

Tech stack: Java, Linux, Gson

Repo: https://github.com/ibrahimanees/sync-app/

Sync easily

Sync App allows users to easily sync two Linux directories. Sync App deals with aspects such as deletions, re-additions, updated files and deletions. Version history of files is mainted using a hidden sync file (.sync).

Run instructions

1) Place a2.jar and sync in the same Linux directory.
2) Open a terminal in the same directory and enter the following command: chmod u+x sync
3) Run the application by entering the following command: ./sync dir1 dir2
