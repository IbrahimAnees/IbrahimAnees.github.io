Sneaker Hub

Full-stack Android app that allows users to buy/browse sneakers.

Tech stack: Android Studio (Java), Firestore database, Figma

Repo: https://github.com/ibrahimanees/sneaker-hub/

More than a sneaker app.

Sneaker Hub aims to provide a community-like experience for all sneaker-lovers. Featured sneakers are ordered by ratings of users world-wide.

Browse by category.

Sneaker Hub allows users to browse sneakers by four different categories (brands).

All the details you need.

Each sneaker's price, colours, sizing, description, details and reviews are fetched from a live database. Users are able to rate sneakers out of five stars.

Find what you want.

Sneaker Hub's advanced search allows you to use multiple filters such as brand, colour and price. Hashmaps are used to quickly search through the database.

Save your favourite ones.

Sneaker Hub lets each user add or remove sneakers from their personalised favourites list.
