Sharketplace

Full-stack web app that allows users to buy and sell digital assets.

Tech stack: MERN stack (Mongo DB, Express, React, Node.js), Socket.io, Stripe, Google OAuth2

Repo: https://github.com/ibrahimanees/sharketplace/

Enabling users to buy and sell digital assets

Sharketplace allows users to buy and sell four categories of digital assets: images, videos, music and services. Users have the ability to list an asset and earn income: users can upload various file types for their assets. Users can also buy other assets.

Real-time chat system with Socket.io

Sharketplace comes with complete chat/private messaging functionality between users. SharkBot is a bot that automatically messages users to notify them when another user has purchased their product.

Technologies used

  • MERN stack
  • Socket.io: Chat/private messaging functionality
  • Stripe: Allowing users to make and accept payments
  • Google OAuth2 & Passport.js: Handling sign-in and authentication
  • Bootstrap: For frontend styling
  • React Paginate: Library for pagination
  • React tsParticles: Library for particle effect on landing page
  • Moment.js: Library for formatting timestamps
