YouTube Audio Downloader

Download multiple YouTube videos as MP3 files.

Tech stack: Python, yt-dlp, FFmpeg/FFprobe

Repo: https://github.com/ibrahimanees/youtube-audio-downloader/

Skip downloading one at a time

YouTube Audio Downloader allows users to simultaneously download multiple YouTube videos as MP3 audio files, saved with custom output names. Other services only allow users to download one file at a time, however YouTube Audio Downloader allows as many files as you want at once.

Populate urls.txt

When populating the urls.txt file, each line should contain the URL followed by the desired output name. The URL and title should be separated by a whitespace character.

Run instructions

1) Ensure the following prerequisites are installed: python3, yt-dlp, ffmpeg and ffprobe
2) Place download.py in a folder
3) Create a new file called urls.txt in the same folder and populate it with URLS and names
4) Using command line, run the following command: py download.py
